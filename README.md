# Anshu Portfolio

A modern personal portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Responsive one-page portfolio layout
- Smooth animated sections
- Project showcase
- Contact section
- Custom cursor experience

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion (if used in the project)

## Project Structure

- src/components — reusable UI sections
- src/App.tsx — main app entry
- src/index.css — global styles
